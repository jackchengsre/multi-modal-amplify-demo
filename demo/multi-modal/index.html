How Multi-Modal Analysis Works

1

Data Collection

Jack Cheng's SRE Copilot collects textual logs, numerical metrics, and visual dashboard images from multiple sources.

2

Modal-Specific Processing

Each data type is processed by specialized AWS Bedrock foundation models optimized for that modality.

3

Cross-Modal Correlation

The system identifies relationships between events across different data modalities, connecting log entries with metric spikes and dashboard anomalies.

4

Visual Pattern Recognition

AWS Bedrock's vision capabilities identify patterns, anomalies, and trends in dashboard visualizations that might be missed by humans.

5

Unified Analysis

All insights are combined into a comprehensive root cause analysis that leverages the strengths of each data modality.
